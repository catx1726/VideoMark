核心功能

不只是书签，是带时间坐标的视频记忆

一键标记

按下 Ctrl+Shift+L，瞬间记录当前视频时间点。无需暂停，不打断观看节奏。

智能截图

直播自动截图留存（事后无法回看）；点播视频可选截图策略，省空间也能留回忆。

精准回跳

点击侧边栏标记，瞬间回到对应视频、对应时间点。3 层 URL 匹配，绝不会找错视频。

直播识别

自动检测直播流，特殊标记并提示无法跳转。截图留存，让精彩画面不再流逝。

常见问题

标记的数据存在哪里？
100% 本地存储在浏览器中，无需注册，不采集任何数据。可选通过 GitHub Gist 实现多端同步。
直播标记为什么无法跳转？
直播是实时流，标记的时间点只对应当前直播时刻，直播结束后该时间点已不存在。系统会自动为直播标记截图，让你事后还能看到那个画面。
快捷键冲突怎么办？
可在浏览器扩展管理页面的「键盘快捷键」中自定义，或在 VideoMark 设置页面直接修改。
支持哪些浏览器？
Firefox（推荐）和 Chrome / Edge 等 Chromium 内核浏览器。采用 Manifest V3 标准。
